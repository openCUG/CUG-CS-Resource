# CUG-CS-Resource

Index of course materials for Computer Science and Technology at China University of Geosciences (Wuhan).

[中文文档](README.md)

## Overview

This project aims to organize and share course materials and code from the undergraduate period.

The materials organized previously (during freshman year) were somewhat messy. To facilitate management and retrieval, this repository will serve as an **index page**, providing links to specific course repositories instead of storing all files directly.

This maintains a clear structure and facilitates maintenance for specific courses. Maintainers can focus on their own projects and only need to provide links and update this `README.md` document when summarizing.

## Co-maintenance

We cannot guarantee the completeness and validity of the materials on our own, so *Issues or Pull Requests to add more materials are welcome!*

At the same time, we remind everyone to pay attention to:

- Do not leak personal information. If you find any leaks, please submit an *Issue or Pull Request* to remind us.
- Strictly abide by the LICENSE.
- Repositories containing code are responsible for code quality.
- ...
