# CUG-CS-Resource

[English Document](README-en.md)

中国地质大学(武汉)计算机科学与技术专业课程资料索引.

## Overview

本项目旨在整理和分享本科期间的课程资料与代码.

此前(大一时期)整理的资料较为杂乱，为了方便管理和检索，本仓库将作为**索引页**，提供指向各个具体课程仓库的链接，不再直接存储所有文件.

这样可以保持结构的清晰，也便于针对特定课程进行维护. 维护者平时可以关注自己的项目, 汇总时只用给出链接+更新本`README.md`文档即可 .

## Co-maintenance

仅靠我们不能保证资料的完整性和有效性 因此 *欢迎提交 Issue 或 Pull Request 补充更多资料！*

同时我们提醒大家注意 :

- 不要泄露个人信息 如果发现泄露之处也可以提交 *Issue 或 Pull Request* 提醒我们
- 严格遵守 LICENSE 
- 含有代码的仓库要对代码质量负责
- ...
